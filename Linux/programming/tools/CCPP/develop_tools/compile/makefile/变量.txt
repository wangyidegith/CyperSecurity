自定义变量：
	target（这是我们自定义的变量名）=app（终极目标名）



makefile维护的变量（大写，因而自定义变量默认小写）：
	CC	默认值是cc，我们一般将其修改为gcc
	CPPFLAGS          预处理参数，如：-I
	CFLAGS          编译参数，如：-Wall -g -c
	LDFLAGS          链接参数，如：-L -l



这里有一个公式%，当我们给出终极目标的小目标依赖的时候，我们就不要再费力去写小目标了，只需要写：
	%.o:%.c
		gcc -c $< -o $@
就可了



上面用到的$<和$@是自动变量：
	$<	规则中的第一个依赖
	$^	规则中的所有依赖
	$@	规则中的目标

### 多态
1 静态：重载（C语言是不支持函数重载的）和模板
2 动态：虚函数和纯虚函数
包含虚函数的类通常被称为多态类或多态基类或虚基类，纯虚函数所在的类被称为抽象类。
区别：
虚函数：
class Base {
public:
    virtual void func() { // 虚函数
        // 可以有实现
    }
};
class Derived : public Base {
public:
    void func() override { // 重写了基类的虚函数
        // 新的实现
    }
};
纯虚函数：
class Base {
public:
    virtual void func() = 0; // 纯虚函数，没有实现
};

class Derived : public Base {
public:
    void func() override { // 必须实现基类的纯虚函数
        // 实现内容
    }
};

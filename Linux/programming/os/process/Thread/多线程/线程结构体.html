成员 可取值 默认值 相关操作函数
stacksize 任意正整数 系统相关，通常为 PTHREAD_STACK_MIN pthread_attr_getstacksize() 和 pthread_attr_setstacksize()
detachstate
  • PTHREAD_CREATE_JOINABLE（非分离状态）
  • PTHREAD_CREATE_DETACHED（分离状态）
 PTHREAD_CREATE_JOINABLE pthread_attr_getdetachstate() 和 pthread_attr_setdetachstate()
stackaddr 任何指针值 NULL pthread_attr_getstackaddr() 和 pthread_attr_setstackaddr()
stackguard 任意正整数 系统相关 pthread_attr_getguardsize() 和 pthread_attr_setguardsize()
schedparam sched_param 结构体 系统相关 pthread_attr_getschedparam() 和 pthread_attr_setschedparam()
schedpolicy 任意一个调度策略，如 SCHED_FIFO、SCHED_RR、SCHED_OTHER 等 SCHED_OTHER pthread_attr_getschedpolicy() 和 pthread_attr_setschedpolicy()
inheritsched
  • PTHREAD_INHERIT_SCHED
  • PTHREAD_EXPLICIT_SCHED
 PTHREAD_INHERIT_SCHED pthread_attr_getinheritsched() 和 pthread_attr_setinheritsched()
scope
  • PTHREAD_SCOPE_SYSTEM（系统级线程，可以被所有进程访问）
  • PTHREAD_SCOPE_PROCESS（进程级线程，只能被创建此线程的进程访问）
 PTHREAD_SCOPE_SYSTEM pthread_attr_getscope() 和 pthread_attr_setscope()

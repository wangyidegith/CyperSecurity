#include <sys/types.h>
#include <sys/stat.h>
int mkfifo(const char *pathname, mode_t mode);   // 该函数的作用是创建一个fifo文件

返回值：成0败-1
参数：pathname是fifo文件的路径（如果没有提前创建则会新建一个fifo文件，如果有了则不产生任何影响，但是使用mkfifo命令可不能重复创建），mode和chmod的一样

在使用mkfifo函数时，要注意三点：
①管道是半双工的，这一点对于命名管道来讲也是这样，②③是二者不同的地方
半双工意即命名管道只能实现数据的单向传输
虽然命名管道可以由多个进程同时读写，但是每个进程只能在同一时间内执行读或写操作。如果同时进行读写操作，就会死锁
②命名管道可以用于任何进程间通信
③命名管道文件一般位于/tmp中，它是一个实实在在的存在硬盘上的文件，但是它不是一个普通文件

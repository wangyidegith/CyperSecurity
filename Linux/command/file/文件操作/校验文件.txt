md5sum [OPTION] <FILE（可以有多个）>

option：
-b 或 --binary：以二进制模式读取文件。
-c 或 --check：从文件中读取 MD5 校验和并校验文件的完整性。
-t 或 --text：以文本模式读取文件。

-b 或 --binary
（1）当使用 -b 或 --binary 选项时，sha256sum 命令会以二进制模式读取文件。
（2）这意味着，sha256sum 不会对文件内容进行任何特殊处理，而是将文件按照它们实际的字节进行处理。
（3）这在处理非文本文件或包含特殊字符（如NULL字符）的文件时很有用，因为在文本模式下，某些特殊字符可能会被处理成换行符或者被忽略，导致计算的散列值不准确。
-t 或 --text
（1）当使用 -t 或 --text 选项时（这是默认选项），sha256sum 命令会将文件以文本模式读取。
（2）在文本模式下，sha256sum 会根据系统的文本文件规范处理文件内容。通常这意味着会处理换行符、回车符等特殊字符，使得文件在不同系统上的行尾符号保持一致性。
（3）但是，这也可能会导致一些特殊字符的处理不同，从而影响计算出的散列值。

-c的用法：
md5sum <file>
md5sum -c <包含md5sum <file>输出结果的文件>

chmod命令：
-R：递归操作
不要再无脑sudo了，你要看这个文件的所有者是谁，是你你就不用sudo了，也就是说，使用文件上传漏洞上传一个agent上去后，如果你发现这个文件的所有者还是www-data，那么直接chmod就行了，还提个dir的权。







数字法：
chmod [u|g|o] 文件名
    u是文件所有者
    g是文件所属组
    o是其他人
    r 4
    w 2
    x 1







字母法：
chmod [u|g|o][+|-][r|w|x|s] 文件名







提权相关：
0 前提：S权限只适用于二进制文件（chmod的前提是文件拥有者必须拥有该二进制文件的X权限或者说该文件的第三个权限位必须是X）。
1 SUID（Set User ID），SGID（Set Group ID）
功能：执行后仅在执行期间拥有文件所有者的权限，但是你要注意，不是谁都能执行该文件而是能执行该文件的人才能以文件的拥有者身份执行该文件。
2 表示：
（1）字母法用s表示；
（2）数字法：
若想设置SUID，则4ugo；若想设置SGID，则2ugo。
3 体现：设置后，s会体现在第三和六位上，不可能出现在第九位上，该权限仅适用于文件所有者（SUID）和所属组（SGID）。
